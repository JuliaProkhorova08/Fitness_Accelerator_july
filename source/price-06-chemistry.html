Химические компоненты VALKARN, SUPER VALKARN G, SEALINER, TERMO LIQUID, BEAD SEALER, LIQUID BUFFER
для ремонта шин и камер

SUPER VALKARN G - вулканизирующий клей. Используется для ремонта всех видов покрышек. Предназначен для холодной вулканизации, не требует нагрева.
Клей VALKARN используется для ремонта всех видов камер от велосипедных до спецтехники. Обладает высокими вулканизирующими свойствами. Предназначен для холодной вулканизации.
TERMO LIQUID - клей для горячей вулканизации. Используется для ремонта разнообразных типов шин.

Герметик борта BEAD SEALER создает прочное уплотнение между бортом шины и ободом диска.
SEALINER - герметик бескамерного слоя. Используется только при ремонте бескамерных шин. Восстанавливает герметичный слой покрышки, поврежденный при зачистке.
Обезжириватель LIQUID BUFFER используется при небольших повреждениях. Его использование избавляет от необходимости зачищать и обезжиривать поврежденную поверхность при ремонте.
